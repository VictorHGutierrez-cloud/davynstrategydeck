Secure, modern cloud architecture
Cloud infrastructure
All our services run in the cloud. We do not host or operate our own 
routers, load balancers, DNS servers, or physical servers. All client 
data is stored on Amazon Web Services (AWS) servers in Frankfurt, 
Germany, a cloud web services platform that ensures maximum 
security. Companies like Netﬂix and Airbnb trust AWS to manage the 
data of millions of users. The Amazon Web Services data center is 
protected by three physical layers of security. Additionally, the 
facilities are impact resistant and accessible only with a 
non-transferable personal card and PIN. You can read more about 
security practices here: AWS
Network-level security monitoring and protection
Our network security architecture consists of multiple security zones. 
We monitor and protect our network to ensure no unauthorized 
access occurs through: Virtual Private Cloud (VPC), a ﬁrewall that 
monitors and controls both incoming and outgoing network trafﬁc.
