Network-level security monitoring and protection
We protect our users from data leaks by monitoring and blocking brute force attacks.
Single sign-on (SSO) is available using Google, Microsoft, and LinkedIn accounts.
Permission-based access control is offered across all our accounts and allows users to deﬁne 
permissions.
We use AWS Cognito, so multi-factor authentication is supported by default.
We utilize Github security tools to receive vulnerability alerts. The security team routinely applies 
patches.
We conduct quarterly access rights reviews on our critical applications, including steps such as reviewing 
authorizations, generic accounts, and ensuring that access for terminated employees is removed.
